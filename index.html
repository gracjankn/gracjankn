Czyszczenie komputerów 🧽

Które systemy czyszczę? 🧐

  • Windows 11
  • Windows 10
  • Windows 8.1
  • Windows 8
  • Windows 7

Co w skrócie robię? 👀

  1. Odinstalowuję zbędne programy.
  2. Usuwam pliki tymczasowe.
  3. Wyłączam programy niepotrzebnie otwierające się po włączeniu komputera lub działające w tle.
  4. Instaluję zaległe aktualizację programów.
  5. Skanuję system pod kątem obecności wirusów.
  6. Zmieniam niektóre ustawienia systemu, pod kątem wydajności.
  7. Blokuję reklamy i skrypty śledzące (przy okazji internet szybciej działa).
  8. Ustawiam serwery DNS (internet może troszkę szybciej działać).

Jakich narzędzi używam (poza systemowymi)? 🛠

  • TeamViewer do połączenia się z komputerem zdalnie.
  • Bulk Crap Uninstaller do odinstalowania zbędnych programów.
  • BleachBit do usuwania plików tymczasowych.
  • uBlock Origin do blokowania reklam i skryptów śledzących w przeglądarce.
  • O&O ShutUp10++ do wyłączenia inwazyjnych opcji w Windows 10 i Windows 11.
